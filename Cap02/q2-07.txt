2.7 Preencha as lacunas em cada uma das seguintes afirmações.

a) COMENTÁRIOS são utilizados para documentar um programa e aprimorar sua legibilidade.

    Em Java existem três tipos de comentários: comentário de final de linha, comentário tradicional e comentário javadoc.
    Os comentários são sempre ignorados pelos compiladores e portanto não interferem no desempenho de qualquer programa.

    Os comentários de linha começam com os caracteres "//" e transformam todo o restante da linha em que aparecem em um
    comentário. São muito úteis para fazer comentários curtos ou explicar a delacaração de uma variável

    Os comentários tradicionais iniciam com "/*" e finalizam com "*/" e transformam todas as linhas entre a marcação de
    abertura e fechamento em comentários.

    Os comentários javadoc iniciam com "/**" e finalizam com "*/", são utilizados para gerar uma documentação automática
    para o código que está sendo escrito.

b) Uma decisão pode ser tomada em um programa Java com um(a) INSTRUÇÃO DE SELEÇÃO IF.

    A instrução if é imediatamente seguida por parênteses e uma expressão booleana, que devee resultar em verdadeiro ou 
    falso. O if também permite detemrinar um trecho de código para executar no caso de a expressão ser verdadeira e outro
    trecho para executar caso a expressão seja falsa.

    Embora nem sempre seja obrigatório utilizar as chaves logo após a expressão condicional, fazer isso facilita a 
    legibilidade do código. Quando houver apenas uma única instrução após a expressão condicional, o uso de chaves de
    abertura e fechamento será opcional. Caso não sejam utilizadas as chaves, é recomendado recuar o corpo da instrução if.

c) // começa em um comentário de fim de linha.

    Os comentários de linha começam com os caracteres "//" e transformam todo o restante da linha em que aparecem em um
    comentário. São muito úteis para fazer comentários curtos ou explicar a delacaração de uma variável

d) ESPAÇOS, REUCOS e LINHAS EM BRANCO são chamados espaço em branco.

    Espaços, recuos e Linhas em Branco são boas ferramentas para organizar o código e melhorar a legibilidade.

e) PALAVRAS-CHAVE são reservadas para uso pelo Java.

    Palavras-chave são sempre escritas com letras minúsculas e tem significados específicos para o Java. Seguem alguns 
    exemplos de palavras-chave: package, import, class, main, private, public, protected.

f) Aplicativos Java iniciam a execução no método MAIN.

    Aplicativos Java de linha de comando sempre iniciam através do método main e pelo fato de ser o ponto de entrada do
    programa, só pode haver um único método main por programa.

g) PRINT, PRINTLN E PRINTF exibem informações em uma janela de comando.

    print apenas imprime o argumento que foi passado e não reposiciona o cursor, fazendo com que a próxima saída 
    prossiga de onde parou. O println é muito semelhante ao print, com a diferença que reposiciona o cursor de saída no
    começo da próxima linha. O printf é o método de saída mais versátil pois permite estabelecer um formato de saída e 
    substituir partes da string  por  variáveis, deixando a saída mais dinâmica.
